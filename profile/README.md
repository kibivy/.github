# Hello!

Hello, this is Redder (A.K.A. KiBivy).
This organization and the repositories associated with it were created solely for archiving purposes.
(This is because older versions may occasionally be deleted from the Google Drive download section.
If the files are deleted from Google Drive, the voice banks can be redistributed.)

To download the voice banks, please check the releases in each character's repository.
You may ignore the compressed file uploaded to the source code, as it contains the original recording files.
(For those unfamiliar with the GitHub system: Releases are located on the right side when you enter the repository.
You can view all the versions I have uploaded on the release page.)

P.S. The English translation was written using a translation tool, so there may be some awkward expressions.
